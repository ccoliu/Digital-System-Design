This file can run on unbuntu environment.

How to use:
This program needs to provide four arguments:
>> PROGRAM_NAME input.kiss output.kiss output.dot
PROGRAM_NAME must be the name of this program (B11001035)
input.kiss must be the name of the input kiss file (you can name anything as you want)
output.kiss must be the name of the output kiss file (same, you can name anything as you want)
output.dot must be the name of the output dot file (also, you can name anything as you want)
Ex: >> ./B11001035 input.kiss output.kiss output.dot

PS: The input file must be in the same directory as this program, make sure there is ALL in English in the directory.
