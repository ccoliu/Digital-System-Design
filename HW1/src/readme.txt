This file can run on unbuntu environment.

How to use:
This program needs to provide three arguments:
>> PROGRAM_NAME input.pla output.dot
PROGRAM_NAME must be the name of this program (B11001035)
input.pla must be the name of the input file (you can name anything as you want)
output.dot must be the name of the output file (same, you can name anything as you want)
Ex: >> ./B11001035 input.pla output.dot

PS: The input file must be in the same directory as this program, make sure there is ALL in English in the directory.
